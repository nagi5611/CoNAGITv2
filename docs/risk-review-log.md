# リスクレビュー記録ログ（テンプレ）

実装計画書 §7 のリスク登録簿とは別に、**運用でのレビューやインシデント後の振り返し**を一行ずつ残す場合に使う。記載する日付・参加者・結論は **実際に実施した事実のみ** とする。

## 記録テンプレ（コピー用）

```
## （検討日またはレビュー実施日を記載）

- **参加者**: （役割と氏名またはチーム名）
- **トリガー**: （定期レビュー / インシデント ID / 監査 等）
- **見た範囲**: （例: §7 の R01〜R20、またはサービス全体）
- **結論の要約**: （リスク行の更新があれば PR 番号）
- **フォローアップ**: （チケット・次回日）
```
